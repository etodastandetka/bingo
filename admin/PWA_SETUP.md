# PWA настройка для админки

Админка поддерживает установку как Progressive Web App (PWA).

## Генерация иконок

Для создания иконок выполните:

```bash
npm install --save-dev canvas
npm run generate-icons
```

Или используйте онлайн конвертер для создания PNG иконок из SVG.

## Установка как приложение

1. Откройте админку в браузере
2. В адресной строке появится иконка установки
3. Нажмите "Установить" - приложение появится на рабочем столе

## Файлы PWA

- `public/manifest.json` - манифест приложения
- `public/sw.js` - service worker для офлайн работы
- `public/icon-192.png` - иконка 192x192
- `public/icon-512.png` - иконка 512x512
