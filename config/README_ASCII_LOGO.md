# ASCII Логотип для консоли

Этот модуль позволяет настроить ASCII-арт логотип, который будет отображаться при запуске приложений в консоли.

## Где находится конфигурация

### 1. Telegram Bot
**Файл:** `telegram_bot/config.py`

Переменная `ASCII_LOGO` содержит логотип для Telegram бота. Логотип выводится автоматически при запуске бота.

### 2. Payment Site
**Файл:** `payment_site/app.py`

Переменная `ASCII_LOGO` содержит логотип для Payment Site. Логотип выводится при запуске Flask приложения.

### 3. Server Monitor
**Файл:** `server_monitor.py`

Переменная `ASCII_LOGO` содержит логотип для Server Monitor. Логотип выводится при запуске скрипта.

### 4. Общий конфиг (опционально)
**Файл:** `config/ascii_logo.py`

Общий модуль для всех приложений (если нужно использовать один логотип везде).

## Как изменить логотип

1. Откройте нужный файл (см. выше)
2. Найдите переменную `ASCII_LOGO`
3. Замените содержимое на свой ASCII-арт

Пример:
```python
ASCII_LOGO = """
╔═══════════════════════════════════════╗
║                                       ║
║          YOUR LOGO HERE               ║
║                                       ║
╚═══════════════════════════════════════╝
"""
```

## Генераторы ASCII-арта

Можно использовать онлайн-генераторы для создания ASCII-арта:
- https://www.ascii-art.de/
- https://asciiart.website/
- https://patorjk.com/software/taag/

## Использование

Логотип выводится автоматически при запуске приложений. Если нужно вывести логотип вручную:

```python
from config import print_logo

print_logo()
```

Или получить логотип как строку:

```python
from config import get_logo

logo_text = get_logo()
print(logo_text)
```
