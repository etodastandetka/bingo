# Установка админки как PWA приложения

Админка поддерживает установку как Progressive Web App (PWA), что позволяет установить её на рабочий стол как нативное приложение.

## Как установить

### Chrome / Edge (Windows, Linux, Mac)

1. Откройте админку в браузере
2. В адресной строке нажмите на иконку установки (или меню → "Установить Bingo Admin")
3. Подтвердите установку
4. Приложение появится на рабочем столе и в меню приложений

### Safari (iOS / iPadOS)

1. Откройте админку в Safari
2. Нажмите кнопку "Поделиться" (Share)
3. Выберите "На экран «Домой»" (Add to Home Screen)
4. Подтвердите установку
5. Иконка появится на главном экране

### Firefox (Android)

1. Откройте админку в Firefox
2. В меню выберите "Установить" или "Добавить на главный экран"
3. Подтвердите установку

## Возможности PWA

- ✅ Работает как нативное приложение
- ✅ Иконка на рабочем столе
- ✅ Запуск в отдельном окне
- ✅ Офлайн кеширование (базовое)
- ✅ Быстрый доступ без открытия браузера

## Технические детали

- **Manifest**: `/manifest.json`
- **Service Worker**: `/sw.js`
- **Иконки**: `icon-192.png`, `icon-512.png`
- **Theme Color**: `#0ea5e9`

## Регенерация иконок

Если нужно обновить иконки:

```bash
npm run generate-icons
```

Или используйте `public/generate-icons.html` в браузере.
