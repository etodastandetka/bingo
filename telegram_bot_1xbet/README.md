# Бот для 1xBet

Этот бот автоматически работает только с казино 1xBet. Пользователям не нужно выбирать казино - по умолчанию используется 1xBet.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` в папке `telegram_bot_1xbet`:
```
BOT_TOKEN_1XBET=ваш_токен_бота
API_BASE_URL=http://localhost:3001/api
PAYMENT_SITE_URL=http://localhost:3003
```

Или используйте тот же токен, что и для основного бота:
```
BOT_TOKEN=ваш_токен_бота
API_BASE_URL=http://localhost:3001/api
PAYMENT_SITE_URL=http://localhost:3003
```

## Запуск

```bash
python bot.py
```

## Отличия от основного бота

- Автоматический выбор 1xBet при депозите и выводе
- Пользователям не нужно выбирать казино из списка
- Все остальные функции работают так же, как в основном боте
