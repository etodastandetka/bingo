# Payment Site

Локальный сайт для оплаты пополнений с QR кодом и таймером

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите сервер:
```bash
python app.py
```

Сайт будет доступен по адресу: http://localhost:3002

## Функционал

- Отображение суммы пополнения
- Генерация QR кода для оплаты
- Таймер обратного отсчета (5 минут)
- Кнопки банков для оплаты (Mbank, О деньги, BAKAI, MEGApay)
- Автоматическая генерация QR кода через API

## API

- `GET /pay?amount=200.50&qr=hash&request_id=123` - страница оплаты
- `POST /api/generate-qr` - генерация QR кода
